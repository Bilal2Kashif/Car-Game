
🚗 Car Sprite Game (SFML - C++)

This is a 2D endless car racing game built using C++ and SFML, where the player controls a car and must avoid incoming enemy vehicles while collecting coins to increase their score.

🎮 Gameplay
1.The player controls a car moving left and right across multiple lanes.
2.Enemy cars spawn randomly and move downward at high speed.
3.Colliding with an enemy car results in game over.
4.Coins appear randomly on lanes — collecting them increases your score.
5.The game continues infinitely until the player crashes.
🕹️ Controls
1.ENTER → Start the game
2.LEFT ARROW → Move left
3.RIGHT ARROW → Move right
4.P → Pause / Resume
5.ESC → Exit game
✨ Features
1.Fullscreen responsive gameplay based on screen resolution
3.Smooth lane-based movement system
4.Random enemy spawning system
5.Coin collection and scoring system
6.Collision detection with adjusted hitboxes for better gameplay
7.Pause and start screen functionality
8.Continuous road animation for realistic movement
🧠 Concepts Used
1.Object-oriented design using structs (EnemyCar, Coin)
2.Collision detection using bounding boxes
3.Vectors for dynamic object management
4.Game loop and real-time input handling
5.Sprite rendering and texture management in SFML
🖼️ Assets
1.Car sprite
2.Enemy car sprite
3.Coin sprite
(Make sure to update file paths or place assets correctly before running)
⚙️ Requirements
1.C++ Compiler (G++ / MSVC)
2.SFML Library installed and configured
🚀 How to Run
1.Install SFML and link it with your project
2.Clone this repository
3.Update asset file paths if needed
4.Compile and run the project
